Privacidade

O sistema de abaixo-assinado utiliza o login para atestar se a pessoa é realmente aluno do CEDERJ. Este é o critério que utilizamos para ter a absoluta certeza que 100% das assinaturas são de alunos.

Armazena os seguintes dados:

  • Nome completo
  • Matrícula
  • Curso
  • Pólo
  • Perfil(Aluno/tutor)
  • Código - hash MD5(matrícula+senha+palavra_passe), algoritmo. MD5 não há retorno.
  • Data de adesão
  • Frase

Os dados permanecerão guardados até o término do processo. Após, serão destruídos!

Não há acesso ao banco de dados do CEDERJ, mas sim, acesso a plataforma via extensão cURL do PHP. Essa extensão é similar a um navegador e assim é realizado a validação, direto via HTTP.

Quando o aluno acessa a plataforma via sistema de abaixo-assinado, é checado sua id de usuário. Essa id gera o link do perfil público, para sincronizar a matrícula, o nome, o curso, o pólo e o perfil(aluno).

O perfil privado não é acessado. A senha não é armazenada. É gerado um código único aleatório de 32 caracteres, garantindo mais segurança ao manifestante.

Não utilizamos cookies. A sessão expira em 5 minutos. A frase poderá ser editada 3 vezes para corrigir possíveis erros de digitação.

O Manifesto e seu propósito

O manifesto contra o fim das tutorias presenciais surgiu com o tópico da aluna Queila, de computação do Pólo de Itaperuna, no fórum do CompCEDERJ.

A aluna recebeu um email do coordenador Leandro com as seguintes novas regras:

  • Turma com até 10 alunos - sem tutor presencial
  • Turma com mais de 10 alunos - com 1 ou 2 horas dependendo da disciplina (se é ou não considerada difícil)
  • O tutor terá pelo menos 2 horas (mesmo que tenha apenas 1 disciplina com menos de 10 alunos)

Queila, indignada, deu o pontapé inicial para a realização do abaixo-assinado contra a atitude do CEDERJ em relação as tutorias.

Depois de muita discussão, atingimos um consenso e assim o manifesto foi possível ser criado.

O propósito do manifesto é exibir nossa indignação diante a alteração dos critérios de disponibilidade das tutorias presenciais e cobrar melhorias, para que possamos ter mais qualidade de ensino sem desperdício de recursos públicos.

Defendemos a evolução da tutoria presencial, para que seja mais robusta, mais convidativa para os alunos, e traga benefícios práticos para o nosso aprendizado. Defendemos que a tutoria seja repensada para afastar, de uma vez por todas, o fantasma do corte pela escassez de alunos.

Defendemos o futuro do ensino semipresencial.

Neste momento, estamos exibindo nossa indignação diante a alteração dos critérios de disponibilidade das tutorias.

Para o futuro, esperamos ser mais efetivos na contribuição.

Sobre o sistema de abaixo-assinado

Nosso sistema de abaixo-assinado é construído com tecnologias para web atuais como o ajax e o json. Com isso, temos um bom ganho de performance na transferência de dados.

Possui 2 partes de processamento de dados envolvidos: no cliente e no servidor

No servidor temos o PHP processando somente os dados, sem qualquer manipulação de tag html. É o processamento mais pesado:

  • acesso ao banco de dados
  • roteamento de requisições
  • proteção de acesso
  • validação de campos
  • envio de email
  • retorno de html ao cliente
  • paginação

No cliente temos o Javascript com o framework jQuery. O Javascript processa:

  • executa ajax
  • requisições de dados e html para o servidor
  • processa dados json, tanto de envio quanto de recebimento
  • gerencia os efeitos
  • executa a validação e controle de campos de formulário

O sistema utiliza as seguintes tecnologias: xhtml, css, js-jquery, php | xhtml e css3 válidos

Dicas úteis

Errou ao escrever a frase?
A frase poderá ser editada efetuando o relogin.

Desvincular senha da plataforma do código único?
Realize o procedimento normal da validação. Após inscrito, altere a senha na plataforma mas acesse o sistema com a senha antiga, diferente da plataforma.

Independe de aderir ao manifesto, troque suas senhas periodicamente, não clique em links para sites duvidosos e não exponha seus dados abertamente na rede. Mantenha o antivírus atualizado, acesse locais confiáveis com informações claras a respeito de quais informações são armazenadas pelo site.

Formato de texto do abaixo-assinado

Aos Senhores da Diretoria Acadêmica,

Os abaixo-assinados, brasileiros, residentes e domiciliados no Estado do Rio de Janeiro e demais Estados da Federação, alunos do Consórcio CEDERJ, solicitam de V.Sª, a Não redução da carga horária e a reformulação na aplicação das tutorias presenciais, a fim de dar melhor suporte aos discentes para auxiliá-los no processo de aprendizado, tornando-a mais robusta e atraente para o aluno, e evitar uma iminente evasão de tutores.

Consideramos as tutorias presenciais cruciais para uma plena absorção do conteúdo oferecido. Defendemos a evolução das tutorias presenciais e não a sua redução.

Na certeza de sermos atendidos, encaminhamos esse documento em folhas numeradas e assinadas pelos discentes.
