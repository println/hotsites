Aplicativo ou vírus?

Coisas difíceis de perceber

Design atual

Problema

Ícones são implementados em computação para informar rapidamente ao usuário sobre uma determinada funcionalidade de uma aplicação. Atualmente os ícones podem ser adicionados arbitrariamente em um programa executável. O grande problema é quando são utilizados para induzir o usuário a realizar uma ação insegura.

Possível solução

Uma possível solução é o sistema operacional ou o antivírus informar ao usuário sobre possíveis problemas na execução da aplicação, evitando assim o usuário utilizar a aplicação duvidosa para saber de seus efeitos.

Compilando no eclipse pela primeira vez

Controles com símbolos ambíguos

Design atual

Problema

Um usuário acostumado a usar interface gráfica em todas as suas ferramentas e inexperiente no eclipse sentirá dificuldades para interpretar o símbolo de “Run Web Test” e “External tools”. Tal confusão é ocasionada por os 2 botões terem uma simbologia e cores parecidas, sendo o “External tools” um pouco menor e adicionado de um desenho de uma maleta.

Possível solução

Para solucionar este problema, a sugestão é a modificação do ícone para algo que lembre uma ferramenta externa de forma mais clara ou troque a cor do ícone.

Web Standards e o Internet Explorer

Coisas que juntas não funcionam bem

Design atual

Problema

A imagem exibe esta página rodando no Internet Explorer 9( a versão mais recente é a 10, do Windows 8). É possível notar que o efeito do CSS3 da sombra no texto “Design ruim?” não é exibido. Esse tipo de problema tolhe o usuário de usufruir de alguns recursos disponíveis em sites Web.

Possível solução

A solução paliativa para exibir o recurso de sombra igualmente entre os navegadores é utilizar “filter: Shadow()”, recurso proprietário e fora do padrão Web. A solução definitiva é a utilização de navegadores com um suporte melhor às novas especificações, como: Mozilla Firefox, Google Chrome e o Opera.

Retornar ao ponto de partida

Coisas anatomicamente mal-projetadas

Design atual

Problema

O site apresenta a dificuldade de exibir o retorno à página inicial por falta de um botão início ou home. A tarefa de retornar ao início do site pode ser uma experiência cansativa e desconfortável.

Possível solução

A solução para este problema é adicionar um botão com um ícone de uma casa ou escrever “Home” ou “Início” no mesmo.
